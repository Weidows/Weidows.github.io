# 源目录

- `public` 里面是用于部署的 HTML/css/js 文件
- `source` 这里面存放文章和 Nav 的 markdown 文章
  - `_data` hexo 插件数据
  - `_post` 发布的文章
  - `tags` 这里面的特殊,我把一些 hexo 插件页面放在这里面
  - `categories` 分类和 Git 仓库页面
- `_config.yml` hexo 带的配置文件
- `package.json` 里面记录 hexo 已安装的插件和依赖
