创业方向思考

关于第一波起的创业方向：

大小核策略

  • 冲击中高端品牌形象，实际收割中下游冲量
  • 高产阶级也会参与消费，但耗费对其来说不大
  • 典型案例：奶茶咖啡/Live/旅游/Vtb/直播带货

卷积挖历史

  • 有点怪相的现象，比如 23 年的老二次元最喜欢听的歌很有可能是 13 年附近的
  • 挖掘怀旧情结和新需求结合

成功要素分析

创业成功的几个关键点：

  1. 丰富且适合的人脉
  2. 具备一定的领导规划能力
  3. 战略方向选择正确

Elon-Musk 案例分析

突然冒出来个想法，是在想马斯克如何走到这步时想到的。

很多个早期采访中，马斯克大致也是言辞一致，并不完全 care 如何挣钱，烧光自己的刀乐也要尽可能继续做下去。

他很会操纵或者说引爆舆论（不过人家执行力也是顶级），收购 X 应该也是他布局很久的一步，毕竟 X 本身也是个成熟的项目不至于大亏损（当然也确实存在问题，参见老马-twitter 大裁员），收了 X 是在做提升愿力那一步，给其他项目做资本打火石。

（当时 Tesla 以及后来的 SpaceX 的起色一般，包括现在在做的脑机，放谁身上一想都是未经验证的，风险概率极大的伪需求）

Tesla 发展历程

特斯拉从初创到发展中期经历了几个关键阶段 [1]：

  • 2003-2008 年：推出 Roadster，面临巨大资金压力，2008 年成功 IPO 筹集 2.26 亿美元
  • 2009-2015 年：推出 Model S 和 Model X，建立充电网络，收入显著增长但经历亏损波动
  • 2016-2020 年：推出 Model 3 和 Model Y，目标让电动汽车普及化，成为美国和全球市值最高的汽车制造商

马斯克在特斯拉发展过程中发挥了关键作用，根据福布斯数据，马斯克目前的净资产为 1850 亿美元，是全球最富有的人之一。

SpaceX 与 Neuralink 资金来源

SpaceX 资金来源 [2]：

  • 与 NASA 和其他政府机构的合同
  • 私人投资者的融资
  • 目前估值超过 1000 亿美元

Neuralink 资金来源 [3]：

  • 马斯克自己的投入（超过 1 亿美元）
  • 其他私人投资者的融资
  • 已完成多轮融资，总额达到 4.3 亿美元

YC 投资趋势

2024 年投资重点方向 [4]

YC 2024 年投资主要集中在以下几个方面：

  • AI 和大模型应用：视频、3D、语音等多媒体素材生成，聊天机器人、机器人流程自动化
  • 细分行业 AI 解决方案：在线教育平台、智能问答系统、销售电话转录器
  • 云计算和数据科学工具：自主托管的机器学习模型、一站式数据服务、基于云的数据分析

团队构成特点 [5]

  • 平均人数：2-3 人的小团队，少数项目有 4-5 人或更多
  • 人均资历：大多具有名校背景、大厂经历或连续创业经验
  • 男女分布：男性占绝大多数，女性相对较少

Apple Vision Pro 分析

小三万的价格谁都肉疼，Apple 肯定也清楚隔壁 Meta-quest 的不温不火，以及老黄的 4090 用户端销量。

我预估，这东西跟 00-15 年代的手机一样，即使成本下降了价格也不会下降太多，直到五年后十年后我们觉得三万价格还可以时才买得起。

所以，我感觉哪怕 Apple 真有钱烧的起，做的东西概念性强，实际用起来时，会因为愿力严重不足而发展缓慢。

商业模式对比

马斯克与 Apple 的异同：

相同点：

  • 都是技术明星+流量明星带头

不同点：

  • Apple 大口吃上了互联网和智能机的红利
  • 老马的项目可以说是大冷门

个人思考

这么想来他完全不是在赌，他有足够的号召力和执行力，另一个缺口 Money 一定会被社会补上。

在我看来与老马走相同路子的，是稚晖君，就是很成功的走了。

三要素分析：

执行力，民众愿力，Money 这三个元素一直存在三角关系，没见过谁完全具有这三样。

所以我在想，国内经常见到大流量博主去带货，有没有挖角他们，去做实际产品的（当然，我知道美妆区多得很）。

于是有个怪相：纯做技术的工资挺高的懒得整流量，纯做流量的一丁点技术都不懂，俩枣放不进一个碗，所以如何三仙归洞？

做技术明星，这个思路很有潜力。

市场观察

有些店/产品确实是靠高产支撑起来的，而且是一次性的，vision pro 就是这种。

关于流量：
要花七分精力去打磨一个视频，三份精力量产其他的。
