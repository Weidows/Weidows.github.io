深入环境变量

🎇 深入解析 Window 环境变量.

Scoop

详见🙌Windows 平台软件包管理器选择-Scoop

JDK

通过 Scoop 安装会自动设置

  • 新建JAVA_HOME环境变量值

    • D:\Game\Demo\AdoptOpenJDK\

  • Path 里面添加

    • %JAVA_HOME%\bin

  • JDK 创建 jre: 进入 JDK 根目录管理员模式输入(正常 JDK 自带)

    • bin\jlink.exe --module-path jmods --add-modules java.desktop --output jre
