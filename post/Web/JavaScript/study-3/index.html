对象

  • 几种标准对象

Date

Json

  • 目前网络数据传输最主要格式.

  • Json 的使用

AJAX

  • 三种请求方法:

    • 原生的 js 写法 xhr 异步请求

    • jQuery 封装好的方法$(#name).ajax(“”)

    • axios 请求

  • 挂起,后期学.

面向对象

  • 与 Java 中类-对象有些不同.

原型

  • 通过原型创建对象

  • 太恶心了,所以 ES6 引入 class,参下.

类和继承

  • 跟 Java 几乎一模一样了!

  • 原理上还是利用原型

    • 原型链结构

BOM 对象

  • browser object model

window

  • 这个对象代表浏览器窗口

Navigator

  • Navigator 封装了浏览器的信息

  • 可用来判断平台和浏览器信息,但是不建议使用,这个信息可被修改/伪造.

screen

  • 屏幕信息

location

  • location 代表当前页面的 URL 信息
  • 重要,常用!

document

  • document 代表当前的页面，HTML DOM 文档树

  • 获取/更改标题

  • 获取具体的文档树节点

  • 获取 cookie

    • 劫持 cookie
      • 通过获取 cookie 就可以在另一台机器上在不输密码的情况下登录
      • 比如Bilibili-helper,获取一些 cookie 就可以伪造登录(确认安全情况下妥善利用)

  • history

    • 前进 history.forward()
    • 后退 history.back()

DOM 对象‼

  • document object model 浏览器网页就是一个 Dom 树形结构！

  • 常用操作

    • 遍历 Dom 节点：得到 Dom 节点
    • 更新：更新 Dom 节点
    • 删除：删除一个 Dom 节点
    • 添加：添加一个新的节点

  • 要操作一个 Dom 节点，必须要先获得这个 Dom 节点.
  • 之后尽量都使用 jQuery()

    • 下拉框----select

    • 隐藏域----hidden
    • 密码框----password
