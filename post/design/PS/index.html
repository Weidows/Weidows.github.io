PS 中一些小用法

|字数总计:171|阅读时长:1分钟|阅读量:

快捷键

功能快捷键
复制图层Ctrl + J
调整图层位置和大小Ctrl + T

解锁图层

  • 索引模式无法解锁

    • 图像 -> 模式 ->(索引颜色 ->RGB 颜色)

  • 解锁

    • 双击图层
    • 把锁拖进垃圾桶图标

透明背景

  • 选择 -> 色彩范围，点击取色
  • 选中后按 Delete

定向变色

  • 变色前后对比

  • 首先选择颜色范围, 参考 透明背景

  • 然后右键选择 填充, 或者在菜单里找【图像】——【调整】——【替换颜色】

  • 填入目标色即可.
