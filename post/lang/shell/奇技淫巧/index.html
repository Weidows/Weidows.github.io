XML-error

error on line 228 at column 35: Input is not proper UTF-8, indicate encoding !

这个报错它的意思是有字符并不是 UTF-8 编码, 虽然它报错指明的文件位置是对的, 但是 line/column 有可能不对

比如 jupyter-lab 运行的输出有时候会带有 XML 不支持的特殊符号 [1] (即使 echo $LANG 是 UTF-8)
