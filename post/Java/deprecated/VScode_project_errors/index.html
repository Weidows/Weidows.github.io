еҸ‘иЎЁдәҺ2020-10-16|жӣҙж–°дәҺ2023-12-06|Javadeprecated
|еӯ—ж•°жҖ»и®Ў:279|йҳ…иҜ»ж—¶й•ҝ:1еҲҶй’ҹ|йҳ…иҜ»йҮҸ:
  • й—®йўҳ
  • и§ЈеҶі

й—®йўҳ

  • еңЁжү“ејҖ VScode еҗҺ Java иҷҡжӢҹжңәе·Із»ҸеҗҜеҠЁ,дҪҶжҳҜе№¶жІЎжңүиҜҶеҲ«е·ҘдҪңеҢәйҮҢйқў Maven зҡ„йЎ№зӣ®
  • жүҖд»Ҙ Java еҸӘиғҪжҸҗдҫӣиҜӯжі•дёҠзҡ„й”ҷиҜҜжЈҖжҹҘ,ж— жі•еҹәдәҺйЎ№зӣ®жҸҗдҫӣејҖеҸ‘зҺҜеўғ(еҢ…жӢ¬зј–иҜ‘)
  • жҖ»д№Ӣе°ұжҳҜвҖ”>жҠҘй”ҷ(none-project),иҖҢдё”еӨ§йғЁеҲҶеҠҹиғҪж— жі•дҪҝз”Ё

и§ЈеҶі

  • з»ҸиҝҮдёҖз•ӘжҠҳи…ҫеҸ‘зҺ°иҝҷжҳҜ Project жЈҖжөӢжңәеҲ¶зҡ„й—®йўҳ,еҰӮдёӢ
  • дёҠйқўйӮЈз§ҚеұӮзә§з»“жһ„дјҡдҪҝеҫ— Java иҷҡжӢҹжңәеңЁеҗҜеҠЁжһ„е»әж—¶ Maven йЎ№зӣ®дёҚиў«иҜҶеҲ«.

  • VScode еҸӘиғҪиҜҶеҲ«е·ҘдҪңеҢәжң¬иә«жҲ–иҖ…е·ҘдҪңеҢәеҗ‘еҶ…дёҖеұӮзӣ®еҪ•жҳҜ Maven йЎ№зӣ®,иҝҷж ·жүҚиғҪдҪҝ Java йЎ№зӣ®иҝҗиЎҢзҺҜеўғжҲҗеҠҹжһ„е»әеҗҜеҠЁ.

  • жҢүз…§д»ҘдёӢж–№жі•еҸҜд»Ҙи§ЈеҶіжӯӨй—®йўҳ:

    • и§ЈеҶіж–№жЎҲеұӮзә§дёҖ
    • и§ЈеҶіж–№жЎҲеұӮзә§дәҢ
