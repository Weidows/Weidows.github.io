注意点

  1. markdown 文件写的超过 1K 行后, 每次格式化 / 页面加载的性能损耗是恐怖的, 而且不利于观看, 所以尽量控制 markdown 行数别太多 (我是超过 500~600 行会分页)

  2. 不同平台对 markdown 渲染不同, 有可能会产生格式不对称问题. (例如 GitHub / Gitee / CSDN 对 README 渲染就不一样)
