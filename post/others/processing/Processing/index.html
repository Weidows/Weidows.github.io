环境需求

  • 需要在电脑里下载好 Processing 主程序.

    • 这个 processing 主程序可以由 Scoop 安装(免去配置环境变量).

    • 不然需要在 Path 里添加上主程序目录, 如 D:\\Game\\Scoop\\apps\\processing\\current\\processing-java

  • 在 VScode 里安装 Processing Language 插件, 它支持语法高亮和代码补全
